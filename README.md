# Enhanced Console Calculator

## 📌 Project Overview
The Enhanced Console Calculator is a Java console-based application developed as part of my Java Development Internship. It demonstrates Object-Oriented Programming (OOP), modular design, advanced arithmetic operations, and exception handling.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Power Calculation
- Square Root
- Custom Exception Handling
- Menu-driven Console Interface

## 🛠 Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Exception Handling
- Java Math Library

## 📂 Project Structure

```
enhanced-console-calculator/
│── src/

│   ├── CalculatorException.java
│── README.md

```

## ▶️ How to Run

1. Clone the repository.
2. Open the project in IntelliJ IDEA, Eclipse, or VS Code.
3. Compile all Java files.
4. Run `Main.java`.
5. Select an operation from the menu and enter the required values.
